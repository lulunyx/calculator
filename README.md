# JavaScript calculator
 
This is a simple calculator built with HTML, CSS and Javascript that provides basic functionality that evaluates math expressions:
addition (+);
subtraction (-);
multiplication (*);
division (/);
modular division (%);
clear all (AC);
delete (DEL).
